## 环境变量设置方式
用 Render API 直接 PUT：
```bash
curl -X PUT "https://api.render.com/v1/services/{serviceId}/env-vars" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $RENDER_API_KEY" \
  -d '[
    {"key":"AI_PROVIDER","value":"deepseek"},
    {"key":"DEEPSEEK_API_KEY","value":"***"},
    {"key":"DEEPSEEK_MODEL","value":"deepseek-chat"},
    {"key":"ADMIN_TOKEN","value":"***"}
  ]'
```

⚠️ 注意：body 是数组 `[]` 格式，不是 `{"envVars":[]}`

## 触发部署
```bash
curl -X POST "https://api.render.com/v1/services/{serviceId}/deploys" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $RENDER_API_KEY" \
  -d '{"serviceId":"..."}'
```

## 踩坑记录
1. Render UI 添加环境变量操作不稳定，AppleScript keystroke 在 Chrome 后台时常失效
2. Secret File (/.env) 挂载路径是 /etc/secrets/，但代码 dotenv 默认读不到
3. Render API 的 env-vars PUT 必须用 `[]` 数组格式
4. DEEPSEEK_API_KEY 设置时被 shell 变量截断导致 401 错误，需确认完整 key
5. 环境变量设置后必须重新触发部署才能生效
