## 持久化部署建议

如需正式上线，推荐：

1. **Render.com** - 免费 Node.js Web Service（15分钟无请求后休眠）
2. **Zeabur** - 国内友好，可部署 Node.js + SQLite
3. **Railway** - 免费额度 $5/月，支持 Node.js
4. **VPS（最低 $5/月）** - 阿里云/腾讯云轻量服务器

迁移步骤：
1. 创建 GitHub 仓库并推送代码
2. 在部署平台绑定仓库
3. 设置环境变量（DEEPSEEK_API_KEY）
4. 部署后即可获得固定域名
