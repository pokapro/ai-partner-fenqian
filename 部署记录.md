## 已部署的服务

### 服务端
- Express.js (Node.js 26)
- SQLite (sql.js) 数据库
- DeepSeek API (deepseek-chat)

### 启动命令
```bash
# 终端1：启动 Node 服务
cd ~/pot/ai在线合伙分钱服务
node server/index.js

# 终端2：启动 Cloudflare Tunnel
cloudflared tunnel --url http://127.0.0.1:3000
```

### 环境变量配置
```
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=sk-1b5…e874（来自 ~/.config/deepseek-tui/.env）
DEEPSEEK_MODEL=deepseek-chat
```

## 部署到 Render.com（待完成）

当前阻塞点：无法在浏览器自动化中完成 GitHub 安全验证（需要密码/SSH key 添加权限）。

### 手动步骤
1. 访问 https://github.com/settings/tokens 生成新的 Personal Access Token
2. `git remote set-url origin https://github.com/pokapro/ai-partner-fenqian.git`
3. `git push -u origin main`
4. 在 https://dashboard.render.com 创建 Web Service
5. 连接 GitHub 仓库 `pokapro/ai-partner-fenqian`
6. 设置环境变量：`AI_PROVIDER=deepseek`, `DEEPSEEK_API_KEY=...`
7. 启动命令：`node server/index.js`
8. 部署后即可获得 `*.onrender.com` 域名

### SQLite 数据须知
- Render 免费版重启会丢数据
- V0 阶段数据量不多，可以接受
- 后期迁移 VPS 时直接把 `data/app.db` 文件拷过去即可
